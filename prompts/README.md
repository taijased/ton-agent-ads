# Prompts

This directory contains reusable local engineering guidance.

Use these prompts as reference material for research, design, and implementation work.

The canonical layout is:

```text
prompts/
  architecture/
  domain/
  style/
  testing/
  build/
```

Read the prompt files that match the task and the actual stack in the target app.

Do not assume framework-specific guidance unless the target app actually uses that framework.
